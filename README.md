🐼 Python Data Cleaning Project using Pandas
Project Overview

Real-world datasets are rarely clean. They often contain missing values, duplicate records, inconsistent labels, incorrect datatypes, outliers, unnecessary columns, and formatting issues.

This project demonstrates a complete end-to-end data cleaning workflow using Python and Pandas on a realistic retail sales dataset.

The objective was to transform raw, messy data into a structured, analysis-ready dataset following industry-standard data preprocessing techniques.
